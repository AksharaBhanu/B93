install selenium (java)
1. JDK 
2. IDE (Eclipse)
3. Selenium jar file

Steps
1. open eclipse
2. create a project
3. create a package and a java file, write the code and excute 
4. download selenium jar from https://www.selenium.dev/downloads/
5. create a folder under the java project (lib)
6. copy paste the selenium jar file into the lib folder
7. right click on the jar file --> build path--> add to build path
